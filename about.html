About

What this is (and what it isn’t)

This site is a compact explanation of one idea: small failures can become big outcomes when system conditions allow them to compound. It’s built for clarity and reuse.

How to use it

Start with Patterns to learn the repeatable shapes, then read Scenarios to see the shapes in motion. Use the Checklist when something “minor” feels suspicious.

Scope (deliberately limited)

This isn’t trying to be a comprehensive textbook. It’s a tight reference: patterns, scenarios, and practical questions that help you detect and interrupt cascades.

Why static HTML/CSS

A static site keeps the focus on ideas and structure instead of tooling. It’s fast, readable, and easy to host.
